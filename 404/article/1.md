# golang
## 执行shell指令
```go
package main
import(
    "fmt"
    "os/exec"
)
func main(){
    fmt.Printf("hello world\n")
    cmd1 := exec.Command("touch","go-shell")
    cmd1.Run()
    fmt.Printf("已创造一个文件")
}
```
