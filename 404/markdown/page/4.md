> 安装git

```sh
pkg in -y git
```

> 下载自己的库

```sh
git clone URL.git
```

> 进入仓库，设置git信息

```
git config --global user.name "用户名"
git config --global user.email "登录邮箱"
```

> 添加要上传的文件到这个目录下，然后输入以下指令上传文件

```
git add .
git commit -m "注释"
git push origin master
```
